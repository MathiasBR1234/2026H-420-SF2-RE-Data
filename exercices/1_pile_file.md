# Classe inversée sur les piles et les files

## Objectifs

- Comprendre les concepts de pile et de file
- Rédiger un document markdown pour expliquer ces concepts
- Implémenter une pile et une file en Python

## Instructions

### Étape 1 : Répartition de la classe (5 minutes)

- La classe sera divisée en deux : une moitiée travaille sur les piles, l'autre moitié sur les files.
- Pour chaque moitié, vous formez des groupes de 2 à 3 personnes. Chaque groupe
  fait une duplication de ce dépôt et travaille sur sa propre branche. Une
  personne dans le groupe devra faire une demande de tirage (pull request) pour
  que les autres puissent voir votre travail.

### Étape 2 : Rédaction du document markdown (25 minutes)

- Chaque groupe doit rédiger un document markdown expliquant les concepts de
  pile ou de file, selon le sujet assigné. Votre documenent doit inclure :
  - Une définition claire de la structure de données
  - Les opérations principales et leur noms standard
  - Les avantages et inconvénients de chaque structure
  - Des exemples d'utilisation en sciences ou en informatique

### Étape 3 : Implémentation en Python (25 minutes)

- Chaque groupe doit implémenter la structure de données assignée (pile ou
  file) en Python sous forme d'une classe. Votre implémentation doit inclure
  les opérations principales sous forme de méthodes de classe. Assurez-vous que
  votre code est bien documenté et facile à comprendre.

### Étape 4 : Demande de tirage (pull request) (5 minutes)

- Une fois que votre groupe a terminé la rédaction du document markdown et
  l'implémentation en Python, une personne du groupe doit faire une demande de
  tirage (pull request) pour que les autres puissent voir votre travail.

### Étape 5 : Révision de l'autre moitié de la classe (25 minutes)

- Si vous travaillez sur les piles, vous devez réviser le travail de l'autre
  moitié de la classe qui travaille sur les files, et vice versa. Fournissez
  des commentaires constructifs sur leur document markdown et leur
  implémentation en Python dans la demande de tirage (pull request).
